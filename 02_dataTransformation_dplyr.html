In this module we will use data from gapminder, which tracks demographic data in countries of the world over time. With ?gapminder you can find out more about the gapminder dataset. Notice that the gaminder dataset is stored as a tibble, a modern take on data frames. Use ?tibble to find out more about tibbles.

# install.packages("dplyr")

Excourse: Abbreviations for large numbers

  • Thousand: 1,000 or 1e03 or 1K or 1M. M being the Roman numeral for 1,000.
  • Million: 1,000,000 or 1e06 or 1MM. MM means thousands (times) thousands.
  • Billion: 1,000,000,000 or 1e09 or 1B.
  • Trillion: 1,000,000,000,000 or 1e12 or 1T.

Outlook: Visualizing summarized data with ggplot2

# install.packages("ggplot2")
library(ggplot2)

Initialize ggplot object and declare the input data frame. Try ?ggplot for more information.

ggplot(data = by_continent_year)

Add point geom to create a scatterplot with aesthetics mappings x-axis equals year and y-axis equals mean life expectancy. Try ?geom_point and try ?aes for more information.

ggplot(data = by_continent_year) +
  geom_point(mapping = aes(x = year, 
                           y = meanLifeExp))

Map continent to color aesthetic.

ggplot(data = by_continent_year) +
  geom_point(mapping = aes(x = year, 
                           y = meanLifeExp, 
                           color = continent))

Map total population in billion to size aesthetic.

ggplot(data = by_continent_year) +
  geom_point(mapping = aes(x = year, 
                           y = meanLifeExp, 
                           color = continent,
                           size = totalPopB))
